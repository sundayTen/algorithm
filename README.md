# 알고리즘 스터디

* 장소: 미정
* 미팅: 매주 일요일 10시
* 스터디 방법
  * 2주마다 기본적인 알고리즘 하나씩 멤버가 돌아가며 설명
  * 1주마다 leet code 문제풀이
    * 평균적으로 easy 문제 2개 / medium 문제 2개
  * 각 알고리즘마다 총 8문제씩 풀이
  * 풀고 난 문제들은 레포 정리 및 각자 정리
